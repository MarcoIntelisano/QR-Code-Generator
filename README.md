## :wrench: Dependencies and Installation

- Python >= 3.9.21 (Recommend to use [Anaconda](https://www.anaconda.com/download/#linux) or [Miniconda](https://docs.conda.io/en/latest/miniconda.html))

### :computer: Requirements

- qrcode >= 8.0
- pillow
- PySimpleGUI >= 4.70.1
